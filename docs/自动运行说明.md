launchctl bootout gui/$USER_ID/com.email.assistant

## ⚠️ 重要说明

### 两个版本的区别

虽然两个都是"详细版"，但它们分析的是**不同的邮件**：

| 版本 | 分析范围 | 特点 |
|------|---------|------|
| **23:00 版** | 最近几天的邮件 | 分析未读或近期的邮件 |
| **23:55 版** | **今天 00:00-23:55 的邮件** | **专门总结今天的邮件** |

**建议：** 两个都保留，23:55 的日报专门看今天的新邮件。

## ⚙️ 技术细节

### daily_summary.py 功能
- 每天 23:30 运行
- 获取今天收到的所有邮件
- 使用 DeepSeek AI 详细分析每封邮件
- 自动分类到 8 个类别
- 提取关键信息：链接、DDL、地点、操作
- 英文邮件自动翻译
- 生成美观的 HTML 邮件报告
- 最多处理 30 封邮件（避免超时）

## 🔧 如需修改

### 只保留一个版本

**只保留 23:00 版：**
```bash
USER_ID=$(id -u)
launchctl bootout gui/$USER_ID/com.emailassistant.dailysummary
```

**只保留 23:55 日报：**
```bash
USER_ID=$(id -u)
launchctl bootout gui/$USER_ID/com.email.assistant
```

### 修改运行时间

编辑对应的 plist 文件：
```bash
# 修改 23:55 日报时间
open ~/Library/LaunchAgents/com.emailassistant.dailysummary.plist
```

修改 `<key>Hour</key>` 和 `<key>Minute</key>` 的值，然后重新加载：
```bash
USER_ID=$(id -u)
launchctl bootout gui/$USER_ID/com.emailassistant.dailysummary
launchctl bootstrap gui/$USER_ID ~/Library/LaunchAgents/com.emailassistant.dailysummary.plist
```

## ❓ 常见问题

### Q: 为什么收到两封邮件？

A: 这是正常的：
- 23:00 的邮件分析近期邮件（可能包含前几天未处理的）
- 23:55 的日报专门总结今天新收到的邮件

如果只想要一封，可以关闭其中一个（见上方"只保留一个版本"）。

### Q: 邮件内容不够详细？

A: 现在的版本已经升级为详细版，包含：
- AI 深度分析
- 邮件分类
- 关键信息提取
- 英文翻译
- DDL 紧急度标记

和 23:00 版本一样详细！

### Q: 今天没有收到日报？

A: 检查以下几点：
1. 电脑是否开机？
2. 查看日志：`tail -f logs/daily_summary.log`
3. 手动测试：`python3 daily_summary.py`

# 📧 邮件助手 - 使用说明

## 这是什么？
一个自动帮你分析邮件的 AI 工具！每天自动读取你的邮件，用 AI 分析哪些重要，翻译成中文，然后发摘要给你。

## 你需要准备什么？

### 1. 注册 DeepSeek 并充值
- 访问 https://platform.deepseek.com
- 用手机号注册
- 充值 10-20 元（分析邮件很便宜，10 元够用很久）
- 创建 API Key（一串以 `sk-` 开头的字符）

### 2. 获取 QQ 邮箱授权码
- 登录 QQ 邮箱网页版
- 设置 → 账户 → 开启 IMAP/SMTP 服务
- 获取 16 位授权码（不是登录密码！）

### 3. 设置 Outlook 转发
- 登录学校 Outlook 邮箱
- 设置 → 邮件 → 转发
- 开启"启动转发"，填写你的 QQ 邮箱
- 保存

## 安装步骤

### 第一步：配置
双击运行 **"配置工具.command"**，按提示输入：
- QQ 邮箱地址
- QQ 邮箱授权码
- DeepSeek API Key
- 学校 Outlook 邮箱地址

### 第二步：运行测试
双击运行 **"邮件助手.command"**，如果配置正确，你会收到邮件摘要。

### 第三步：设置定时运行（可选）
想让程序每天自动运行？打开终端，复制粘贴下面命令：

```bash
# 创建每天 23:55 运行的定时任务
cat > ~/Library/LaunchAgents/com.friend.email.plist << 'EOF'
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.friend.email</string>
    <key>ProgramArguments</key>
    <array>
        <string>/usr/bin/python3</string>
        <string>~/邮件助手/final_email_assistant.py</string>
    </array>
    <key>WorkingDirectory</key>
    <string>~/邮件助手</string>
    <key>StandardOutPath</key>
    <string>/tmp/email_assistant.log</string>
    <key>StandardErrorPath</key>
    <string>/tmp/email_assistant.error.log</string>
    <key>StartCalendarInterval</key>
    <dict>
        <key>Hour</key>
        <integer>23</integer>
        <key>Minute</key>
        <integer>55</integer>
    </dict>
</dict>
</plist>
EOF

launchctl load ~/Library/LaunchAgents/com.friend.email.plist
```

## 文件说明

- `final_email_assistant.py` - 主程序（不要直接双击这个）
- `邮件助手.command` - 双击运行这个来分析邮件
- `配置工具.command` - 双击这个来修改配置
- `config.json` - 配置文件（存你的邮箱信息）

## 遇到问题？

**提示"配置文件缺失"**
→ 先运行"配置工具.command"

**提示"认证失败"**
→ 检查 QQ 邮箱授权码是否正确，或 DeepSeek API Key 是否有效

**没有收到邮件摘要**
→ 检查 Outlook 是否成功转发到 QQ 邮箱，QQ 邮箱是否收到邮件

**想停止自动运行**
```bash
launchctl unload ~/Library/LaunchAgents/com.friend.email.plist
```

## 注意事项

⚠️ **不要把 config.json 发给别人！** 里面存了你的邮箱授权码和 API Key

💰 **费用**：DeepSeek 分析邮件很便宜，10-20 元够用很久（约 0.1-0.2 元/次）

📅 **定时运行**：默认每天晚上 23:55 自动运行，可以手动运行"邮件助手.command"随时测试
